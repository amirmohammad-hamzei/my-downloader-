### 🔧 راهنمای استخراج فایل‌های چندبخشی (Multi-Volume ZIP)

فایل‌هایی که با پسوند `.zip` و `.z01`، `.z02` و... ذخیره شده‌اند:

| سیستم‌عامل | روش استخراج |
|-----------|------------|
| **Windows** | روی فایل `.zip` اصلی راست‌کلیک کنید → **7-Zip** → Extract Here |
| **Mac** | روی فایل `.zip` اصلی دابل‌کلیک کنید (**The Unarchiver** خودکار تشخیص می‌دهد) |
| **Linux** | `cat filename.z* > combined.zip && unzip combined.zip` |

**⚠️ نکته مهم:** همه بخش‌ها باید در **یک پوشه** باشند!
